# regex

## 正则基础知识

### 1. 元字符  

* . 匹配除换行符（\n）以外的任何单个字符  
* w 匹配字母、数字、下划线、汉字  
* s 匹配任意空白字符（包括空格、制表符、换页符等）  
* d 匹配数字，匹配单词的开始或结束  
* ^ 匹配字符串的开始  
* $ 匹配字符串的结束  

例子：

```bash
#匹配abc开头的字符串  
^abd

#匹配8位数字的QQ号
^dddddddd$

#匹配以153开头的11位数字手机号  
^153dddddddd$  
```

### 2. 重复限定符

* \* 重复0次或更多次  
* \+ 重复1次或更多次  
* ? 重复0次或1次  
* {n} 重复n次  
* {n,} 重复n次或更多次  
* {n,m} 重复n到m次  

```bash
#匹配8位数字QQ号    
^d{8}$

#匹配153开头11位手机号  
^(153)d{8}$

#匹配身份证号第7到14位（出生日期）    
^d{7,14}$  

#匹配以a开头的，0个或者多个以b结尾的字符串
^ab*$
```  

### 3. 分组  

```bash
#匹配字符串中包含0到多个ab开头
^(ab)*$
```

### 4. 条件或

正则用【|】表示或，当满足分支里任何一种条件时，就会匹配成功

```bash
#匹配手机号中联通的手机（联通号段130/131/132等）
^(130|131|132)d{8}$
```

### 5. 区间

正则提供‘[]’表示区间条件

* [0-9] 限定0到9   
* [a-z] 限定a-z  
* [165] 限定某些数字
